# HexVault — Changelog

Public changelog for [HexVault](https://hexvault.co.uk) — the zero-knowledge password manager and identity access management platform.

## What is HexVault?

HexVault is a zero-knowledge IAM platform built on the principle that your credentials should be mathematically inaccessible to anyone but you — including us.

- **Zero-knowledge architecture** — AES-256-GCM encryption, per-entry key derivation, Argon2id key stretching. Your master password never leaves your device.
- **Identity & access management** — SCIM provisioning, SSO, role-based permissions, just-in-time access, offboarding workflows.
- **HexGuard AI** — the only password manager AI that understands your vault and your organisation's access patterns.
- **Browser extension** — autofill, live TOTP, breach detection, passphrase generator. Available on Chrome, Firefox, and Edge.
- **Watchtower** — continuous breach monitoring, password health scoring, expiry tracking.
- **Multi-platform** — web app, browser extension, desktop app (Windows, macOS, Linux).
