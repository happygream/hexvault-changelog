# Security

HexVault is a security product. We take vulnerability reports seriously and respond promptly.

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities via:
- **Email:** security@hexvault.co.uk
- **Web:** [hexvault.co.uk/responsible-disclosure](https://hexvault.co.uk/responsible-disclosure)

We aim to acknowledge reports within 24 hours and provide a resolution timeline within 72 hours.

## Bug Bounty

Responsible disclosures that result in a fix will be acknowledged in our security hall of fame at [hexvault.co.uk/security/hall-of-fame](https://hexvault.co.uk/security/hall-of-fame).
